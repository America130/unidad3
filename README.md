# unidad3
Este proyecto corresponde al desarrollo de una biblioteca de componentes de Interfaz de Usuario (UI) reutilizables, construidos utilizando CSS avanzado y organizados de manera modular.
La biblioteca incluye componentes visuales y funcionales como botones, tarjetas, formularios, alertas, modales y una barra de navegación responsiva, todos integrados en una página demo funcional.

El objetivo principal es fomentar la reutilización de código, la coherencia visual y la escalabilidad en el desarrollo de sitios y aplicaciones web.
